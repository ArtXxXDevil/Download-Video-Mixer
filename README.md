# 🎬 Download Video Mixer

<img width="1051" height="601" alt="image" src="https://github.com/user-attachments/assets/686d31d9-c4bd-46ab-996c-59f449e6f92b" />


**Download Video Mixer** — это удобное кроссплатформенное десктопное приложение для скачивания видео в высоком качестве и автоматического наложения дополнительных аудиодорожек (например, профессионального или машинного перевода).

Программа создана для тех, кому нужно быстро скачать видео (с YouTube и других площадок) и бесшовно "склеить" его с другой озвучкой, гибко настроив баланс громкости между оригиналом и переводом.

## ✨ Основной функционал

* 📥 **Умное скачивание:** Автоматический анализ ссылки и выбор наилучшего доступного качества видео (вплоть до 8K).
* 🎛 **Аудио-микшер:** Возможность выбрать локальный аудиофайл (mp3, m4a, wav) и наложить его поверх видео.
* 🎚 **Баланс звука:** Встроенные ползунки для настройки громкости (например, приглушить оригинальное видео до 15% и сделать перевод на 100%).
* 🧠 **Умный кэш:** При скачивании программа сохраняет чистый оригинал. Если вы захотите наложить другой перевод на то же видео, программа не будет тратить трафик и время на повторное скачивание, а использует локальный кэш.
* 🚀 **Portable-режим (Автономность):** Программа не требует установки стороннего софта. При первом запуске она сама в фоновом режиме скачает необходимое ядро обработки (FFmpeg) и положит его рядом с собой.
* 💻 **Кроссплатформенность:** Полноценная поддержка Windows (10/11) и macOS (Intel/Apple Silicon).

## 🛠 Технологии и библиотеки

Проект написан на **Python 3** и использует следующие открытые технологии:

* **[yt-dlp](https://github.com/yt-dlp/yt-dlp):** Самый мощный и современный движок для извлечения ссылок и скачивания видео/аудио потоков.
* **[FFmpeg](https://ffmpeg.org/):** Индустриальный стандарт для работы с мультимедиа. Используется для высокоскоростной склейки (мультиплексирования) видео и аудио без потери качества.
* **[CustomTkinter](https://github.com/TomSchimansky/CustomTkinter):** Современная библиотека для создания графического интерфейса (GUI) в Python, обеспечивающая поддержку темных/светлых тем и скругленных элементов дизайна.

## 🚀 Как использовать

Вам не нужно устанавливать Python или работать с терминалом, чтобы использовать программу.

1. Перейдите в раздел **[Releases](../../releases)** на странице этого репозитория.
2. Скачайте последнюю версию для вашей операционной системы:
   * Для Windows: скачайте `.exe` файл.
   * Для macOS: скачайте `.dmg` образ, откройте его и перетащите программу в папку "Программы" (Applications).
3. Запустите приложение. При первом запуске может потребоваться пара минут на автоматическую загрузку ядра FFmpeg.
